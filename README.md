# 📰 Fake News Detection System

A clean, interactive Machine Learning web application built with Python and Streamlit that classifies news articles as either **Real** or **Fake** using a trained classification model.

---

## 🚀 Project Overview
This project utilizes Natural Language Processing (NLP) and Machine Learning to analyze the text of a news article and predict its authenticity. It is designed to assist users in identifying misinformation quickly and easily.

### Key Features
* **Real-time Analysis:** Instantly processes and classifies news text.
* **Interactive Web Interface:** A lightweight, easy-to-use UI built with Streamlit.
* **Machine Learning Backend:** Uses a trained Scikit-Learn model to analyze linguistic patterns common in fake news.

---

## 🛠️ Tech Stack
* **Language:** Python
* **Web Framework:** Streamlit
* **Machine Learning:** Scikit-Learn
* **NLP Vectorization:** TF-IDF Vectorizer

---

## 📁 Repository Structure
* `app.py` - The main Python script that runs the Streamlit user interface.
* `model.pkl` - The pre-trained machine learning model used for predictions.
* `vectorizer.pkl` - The fitted TF-IDF Vectorizer for text preprocessing.
* `requirements.txt` - List of Python packages required to run the application.
* `.gitignore` - Specifies files to ignore when pushing to GitHub.
* `LICENSE` - MIT License protecting this open-source code.
